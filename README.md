
# lochees

Manage and document your tests.

help you to keep track of your test statues from the code itself.

### Run
`lochees dir_path ...args`

### Example: 
`lochees ./e2e --full --output ./debug.log --github-auth secret132`

### Args:

`--output` `-o` Path to the out file.

`--github-auth` `-gha` Your github access token.

`--full` Generate full report.

`--only-warnings` Generate a report that contain only warnings.
